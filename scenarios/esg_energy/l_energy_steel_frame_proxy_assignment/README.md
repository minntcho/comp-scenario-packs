# L-Energy Steel Frame Proxy Assignment

This pack shadows `comp` internal scenario
`l_energy.steel_frame_proxy_assignment.v1`.

It is downstream compatibility evidence only. The prepared bundle records the
accepted proxy-assignment path where Steel Frame has no supplier submission,
the missing lower-tier mass is derived, a proxy factor is bound, and public
projection is authorized by an existing receipt.

The scenario consumes `comp` through `comp.scenario_contracts` and does not
define authority for receipts, replay, or projection.
